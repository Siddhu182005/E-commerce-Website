LUMIS Audio — Premium E-Commerce Product Landing Page

A complete, responsive multi-page e-commerce website for LUMIS, a fictional premium wireless headphone brand.

---

Project Overview

LUMIS Audio is a modern front-end e-commerce project designed for a fictional premium wireless headphone company. The website focuses on delivering a luxury-inspired user experience with responsive layouts, elegant typography, interactive UI components, and smooth user interactions.

This project was developed as a professional front-end web development project for internship portfolio and repository submission purposes.

---

Features Implemented

- Fully responsive design for mobile, tablet, and desktop devices
- Modern navigation bar with active link highlighting
- Mobile hamburger menu
- Product filtering and sorting functionality
- Shopping cart with localStorage persistence
- Toast notification system
- FAQ accordion section
- Contact form with real-time validation
- Scroll reveal animations using IntersectionObserver
- Smooth scrolling navigation
- Accessible semantic HTML structure
- Optimized SVG-based product illustrations

---

Technical Implementation

Responsive Design

- Built using CSS Grid and Flexbox
- Mobile-first responsive approach
- Tested across multiple screen sizes from 320px to 1440px
- Fluid typography using "clamp()"

Form Validation

The contact form includes:

- Required field validation
- Email format validation
- Character count validation
- Inline error messages
- Accessible ARIA attributes
- Success and loading states

Performance Optimization

- Inline SVG graphics used instead of heavy raster images
- Lightweight vanilla JavaScript implementation
- No external frameworks or build tools
- Optimized CSS structure with reusable variables

GitHub Pages Deployment

git init
git add .
git commit -m "Initial project commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/lumis-audio.git
git push -u origin main

After pushing the repository:

1. Open GitHub repository settings
2. Navigate to Pages
3. Select "main" branch
4. Save and deploy
